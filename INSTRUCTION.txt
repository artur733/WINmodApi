EN:

Copy the mods folder and the mods.exe file to the System32 folder and then run the 
command line and type the command: mods This will open the mods system. 
Then enter matrix.
If you get the matrix effect, 
it means that you can install mods by putting the mod files into the mods 
folder and use the name of the mod instead of the matrix

Installation Steps:
1. Copy Mods and Mods.exe to system32
2. Run CMD
3. Enter Mods
4. Enter the modification name (Default presentation modification: Matrix)
5. Done. Now just download the mods you want

PL
Skopiuj folder mods i plik mods.exe do folderu System32, a następnie uruchom
wiersz poleceń i wpisz polecenie: mods Spowoduje to otwarcie systemu modów.
Następnie wprowadź macierz.
Jeśli uzyskasz efekt matrixa,
oznacza to, że możesz instalować mody, umieszczając pliki modów w plikach mods
folder i użyj nazwy moda zamiast matrixa

Kroki instalacji:
1. Skopiuj Mody i Mods.exe do system32
2. Uruchom CMD
3. Wprowadź modyfikacje
4. Wprowadź nazwę modyfikacji (Domyślna modyfikacja prezentacji: Matrix)
5. Gotowe. Teraz po prostu pobierz modyfikacje, które chcesz
